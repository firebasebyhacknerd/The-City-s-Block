# **App Name**: The City's Blocks

## Core Features:

- Property Search & Filtering: Users can search for properties by location, price range, property type, and other key criteria using an intuitive search interface.
- Property Listing Details: Each property will have a dedicated page displaying photos, a detailed description, amenities, location on a map, and agent contact information.
- Property Listing Submission: Authenticated users (agents/owners) can create and manage their property listings, uploading images and inputting property details.
- AI-Powered Description Tool: Assists listers by generating concise and engaging property descriptions based on provided key details (e.g., bedrooms, location, key amenities) using generative AI.
- User Authentication & Profiles: Allow users to register, log in, and maintain a basic profile, enabling them to save favorite listings or manage their posted properties.
- Favorite Listings Management: Logged-in users can save properties to a 'favorites' list for easy access and comparison.
- Contact Inquiry Form: Provide a simple form on each property listing page for potential buyers/renters to send inquiries directly to the property agent/owner.

## Style Guidelines:

- Primary color: A stable and trustworthy deep blue (#3399CC), reflecting the solidity of urban blocks and professional services.
- Background color: A very subtle, cool-toned off-white (#EAF5F9) to provide a clean and expansive feel, typical of cityscapes and blue skies.
- Accent color: A fresh, vibrant teal-green (#4DCCBB), adding a touch of modernity and indicating positive interactions or growth.
- Headline and body text font: 'Inter' (sans-serif) for its modern, clean, and highly readable characteristics, perfect for presenting property information clearly.
- Utilize modern, clean line icons for navigation and features, aligning with a professional, urban, and user-friendly aesthetic. Icons for housing, search, location, and user profiles should be clearly discernible.
- Implement a grid-based layout for property listings to ensure a consistent and organized display of information, mimicking structured city blocks. Prioritize generous spacing and clear hierarchy.
- Incorporate subtle hover effects on listings and smooth transitions for page changes or filter applications, providing a polished and responsive user experience without being distracting.
